Projeto desenvolvido com a ajuda da IA consolidando vendas de E-commerce

# Caracteristicas
    - A empresa foca apenas em fabricação de consoles, deixando a distribuição e venda para terceiros
    - Os produtos são vendidos globalmente

# Objetivos
    - [ ] Consolidar todas as base de terceiros para realizar uma análise
    - [ ] Transformar dados de venda sem informações relevantes para a fabricante
    - [ ] Quais são os produtos mais populares em cada país
    - [ ] Como otimizar o processo de transporte e logística até o momento da venda
